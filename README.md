A simple machine learning project that predicts student performance based on study hours and attendance using Python.
# Student Performance Predictor

This project predicts whether a student will pass or fail based on
study hours and attendance using machine learning.

## Tech Stack
- Python
- Pandas
- Scikit-learn

## How to Run
pip install -r requirements.txt
python student_model.py

## Output
Accuracy score is displayed after model evaluation is
Accuracy: 0.88-1.0
